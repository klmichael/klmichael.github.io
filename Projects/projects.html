Monster Makeover

My course provided us with an "eye exercise" - two eyes in the middle of the screen programmed to follow the cursor's movement. I extended the exercise into a "Monster Makeover" game, giving the user control over the look and movement of the eyes. The user can either click the buttons at the bottom of the screen or, for some features, utilize the arrow keys indicated on the corresponding button.

Bus Tracker Map

We spent the 9th week of the course working with maps to practice asynchronous programming. After completing the assigned challenges, I wanted more practice calling live data and using this data to animate the map. My map allows the user to map all current MBTA bus stops and then to map the locations of all currently running MBTA buses. The bus locations update every 30 seconds.
